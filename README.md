# 🚀 AI Customer Support Automation System

An advanced AI-powered customer support platform that automates email handling using LLMs, sentiment analysis, and Gmail integration.

---

## 🔥 Features

- 📥 Read customer emails directly from Gmail
- 🤖 AI classification (Login, Billing, Technical, Complaint, etc.)
- 😡 Sentiment analysis (Positive / Neutral / Negative)
- 🚨 Smart priority detection (High / Medium / Low)
- ✉️ Auto-generate professional replies using LLM
- 📊 Interactive dashboard with analytics
- ⚡ Real-time processing using Streamlit

## 🧠 Tech Stack

- Python
- Streamlit
- Google Gmail API
- Ollama (LLM - llama3)
- Pandas
- Plotly

---

## ⚙️ How It Works

1. Connects to Gmail inbox
2. Reads incoming emails
3. Uses AI to:
   - Classify email
   - Detect sentiment
   - Assign priority
4. Generates a smart reply
5. Displays everything in a dashboard
